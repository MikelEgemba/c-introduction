Att lära sig C#

Gratis kurser, handledningar, videor och mer om att lära sig programmeringsspråket C#

C# används ofta för att skapa kraftfulla och användarvänliga program för Windows, till exempel kalkylprogram, verktyg eller administrativa system.

C# är det främsta språket i spelmotorn Unity, vilket gör det perfekt för att skapa 2D- och 3D-spel – allt från mobilspel till stora datorspel.

Med hjälp av .NET-ramverket kan C# användas för att bygga moderna, dynamiska webbapplikationer och API:er – perfekt för företag och startups.
